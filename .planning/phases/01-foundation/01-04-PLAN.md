<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTTP client → FastAPI POST /tools/* | Untrusted JSON body enters at the route handler |
| MCP stdio caller → mcp_stdio.py | Tool call arguments arrive from MCP host (e.g. Claude Desktop) |
| CLI user → cli.py argparse | User-supplied topic name enters as a string argument |
| server.py env → uvicorn host/port | KAFKA_MCP_HOST / KAFKA_MCP_PORT from environment |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Tampering | FastAPI POST /tools/describe_topic — topic: str input | mitigate | Pydantic DescribeTopicRequest validates topic is a non-empty string; TopicNotFoundError is raised for unknown topics instead of leaking broker metadata |
| T-04-02 | Information Disclosure | FastAPI 404 error response for TopicNotFoundError | accept | Error detail returns the topic name (already known to the caller); no broker internals, offsets, or credentials exposed in error body |
| T-04-03 | Denial of Service | FastAPI POST /tools/list_topics — large topic list | mitigate | Response is a list[str]; no unbounded deserialization; max broker topics is a broker-side limit; no pagination needed at Phase 1 scale |
| T-04-04 | Information Disclosure | CLI --json output to stdout | accept | CLI is a local tool running as the user; stdout is trusted; no credential output |
| T-04-05 | Spoofing | MCP tool readOnlyHint bypass | mitigate | Defense-in-depth per D-14: readOnlyHint=True on every tool annotation AND the structural assign-based no-commit consumer in plan 01-02; annotation alone is advisory but structure is the true guarantee |
| T-04-06 | Denial of Service | server.py uvicorn without timeout | mitigate | KAFKA_MCP_BOOTSTRAP_SERVERS must be set (fails fast with ConfigError before uvicorn starts); broker poll_timeout default 1.0s from KafkaMcpSettings bounds individual calls |
| T-04-SC | Tampering | fastapi / uvicorn package install | mitigate | Both are canonical, high-download PyPI packages; no [ASSUMED]/[SUS] entries |
</threat_model>
