# OpenSpec for Kafka MCP

This directory contains the OpenSpec specifications for the Kafka MCP project.

## Specifications

- `kafka-mcp-api.json` - OpenAPI specification for the Kafka MCP REST API

## Tools

The project is configured with:
- OpenCode
- Claude Code

## Getting Started

To view the specification:
```bash
openspec view
```

To propose changes:
```bash
/opsx:propose "your idea"
```
